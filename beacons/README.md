##UriBeacon Beacon Implementations

This directory contains examples for various Bluetooth Low Energy
platforms broadcast UriBeacon advertisements. The platforms
are:

* [Bluegiga BLE113](BLE113)
* [RFduino](RFduino)
* [Arduino](Arduino)
* [OpenBeacon](OpenBeacon)
* [Android](Android)
* [linux](linux)
* [nRF51](nRF51)
* [nodejs](nodejs)
* [tessel](tessel)
